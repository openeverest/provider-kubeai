# Maintainers

This file lists the maintainers of `provider-kubeai`, the OpenEverest provider
that turns `Instance` CRs into [KubeAI](https://www.kubeai.org/) `Model` CRs.

| Name | GitHub |
|---|---|
| Atharva Mhaske | [@atharvamhaske](https://github.com/atharvamhaske) |
| Sergey Pronin | [@s-pronin](https://github.com/s-pronin) |

## Responsibilities

Maintainers are responsible for:

- Reviewing and merging pull requests.
- Triaging issues opened via the [issue templates](.github/ISSUE_TEMPLATE/).
- Cutting releases and tagging chart/image versions.
- Deciding on breaking changes to `definition/` (the provider's public schema surface)
  and to the KubeAI `Model` fields this provider owns.
